## Video Demo for Landing page - spanish

### Script

1. La forma de aprender idiomas cambió. Con iu-lern, usa la IA para traducir, memorizar y hablar a tu propio ritmo.
2. Paso uno: recolecta vocabulario real. Mira tus videos favoritos de YouTube con transcripciones y traducciones interactivas. ¿Ves una palabra nueva? Solo haz clic para ver su significado con IA, sin pausar el video.
3. Paso dos: crea tus tarjetas con un clic. No pierdas tiempo escribiendo definiciones; la IA genera automáticamente tu flashcard en la colección exacta que decidas, autocompletando campos con ejemplos de uso real.
4. Paso tres: personaliza tus tarjetas. Entra a tus colecciones y abre cualquier tarjeta. Puedes editar la definición, ejemplos y hasta las conjugaciones en caso de verbos. Puedes editar la información a mano o acelerar el proceso con Inteligencia Artificial. En la pantalla de edición, simplemente le pides a la IA que genere un ejemplo nuevo con su traducción para mejorar tu entendimiento, agrega un sinónimo en un segundo, acepta los cambios y verifica la edición.
5. Paso cuatro: repaso inteligente. iu-lern utiliza un algoritmo avanzado, que calcula el momento exacto en que tu cerebro necesita repasar cada palabra. Para validar tu conocimiento, escribe la traducción en la pantalla. El sistema califica tu progreso de forma inteligente: evaluación completa, parcial o incorrecta. Si fallas una palabra, el algoritmo ajusta el calendario para mostrártela más pronto que las otras.
6. ¿Listo para transformar tu forma de aprender? ¡Regístrate ahora mismo y empieza a usar iu-lern completamente gratis! El idioma que quieres hablar está a solo un clic de distancia.
